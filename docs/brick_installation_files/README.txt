Open Roberta lab EV3 firmware based on leJOS 0.8.1-beta

---Installation guide---

1. Extract lejosimage.zip to SD Card
2. Copy ejre-7u60-fcs-b19-linux-arm-sflt-headless-07_may_2014.gz to SD Card (do NOT extract the content of this file!)
3. Eject SD Card and put it in the brick
4. Start the brick.
5. Wait until installation is finished.
6. Have fun!
